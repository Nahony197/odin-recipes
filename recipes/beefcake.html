Beefcake

Description

Inspired by a nineteenth century recipe, these Beef Cakes are an ingenious and delicious way to use up leftover beef.

In my house, we welcome all kinds of the recipe, and when I have left over beef sometimes, I choose to use them in making the beefcake. Therefore, in using this recipe, you will find an easy way to use the left-over beef meal.

Ingredients

  • Beef
  • Flour
  • Ginger
  • Onion

Steps

  1. Make a cake
  2. Add leftover beef
  3. Bake it
  4. Eat it
  5. Beeeeeeefcake!
